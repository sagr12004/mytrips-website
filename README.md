# MyTrips

**MyTrips** is a beginner-level travel planner web app built for learning and practice. It allows users to explore cities, filter by country, search destinations, view details, and plan trips by specifying dates, budget, and travelers. Trips can be saved for later access.



---

## Features

- Browse popular cities around the world with images.
- Filter destinations by country.
- Search for cities dynamically.
- View details for each destination.
- Plan trips with start/end dates, budget, and number of travelers.
- Save trips securely for logged-in users.
- Responsive design for desktop and mobile.
- User authentication with Supabase (sign-up, login, logout).

---

## Technologies Used

- **HTML5** – Structure of the web pages.  
- **CSS3** – Styling and responsive layouts.  
- **JavaScript** – Dynamic functionality and DOM manipulation.  
- **Supabase** – Backend database and user authentication.  
- **Unsplash API** – City images for cards.
